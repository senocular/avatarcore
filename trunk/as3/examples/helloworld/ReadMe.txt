+------------------------------------+
|         Hello World Example        |
+------------------------------------+
      for Avatar Core Framework
 http://myavatareditor.com/avatarcore 


Description:
	This example demonstrates a simple application of the Avatar Core
	framework using only basic components, somewhat equivalent to Hello
	world! examples used for programming languages but through an Avatar
	Feature rather than text. 
	
Uses:
	Basic Avatar components (Avatar, Feature, Art, AvatarDisplay)
	XML (inline) parsing
	External assets
	
Contents:
	HelloWorld.as
	helloworld.fla (Not required)
	helloworld.html
	helloworld.swf

Versions:
	Language: ActionScript 3.0
	IDE: Flash CS4 (for FLA)
