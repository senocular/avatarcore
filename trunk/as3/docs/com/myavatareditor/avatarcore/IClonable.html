public function clone(copyInto:Object = null):Object

Creates and returns a copy of the current instance.

Parameters

copyInto:Object (default = null) — A variable for internal use only, this value represents the instance of a superclass into which properties of the current class are to be copied into. This allows superclasses to copy private data into subclasses when a subclass creates a copy and gives it to the super clone via: super.clone(copy); When an implementation of clone receives a copy, it uses that instance instead of creating a new instance of its own.

Returns
Object — A copy of the current instance.
